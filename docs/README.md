# ClearPay Docs Index

This repo keeps only the primary entry docs at the root:

- `README.md`
- `QUICK_START_LOCAL.md`

All other Markdown files are grouped under `docs/`:

- `docs/setup/local`
  Localhost, PostgreSQL, XAMPP, and setup verification
- `docs/email`
  Brevo, SMTP, sender verification, and delivery troubleshooting
- `docs/features`
  Feature behavior and implementation notes
  Includes `docs/features/PYTHON_ANALYTICS_INTEGRATION.md` for the PHP-to-Python analytics flow
- `docs/reference`
  API and database reference
- `docs/troubleshooting`
  Runtime issues, migration fixes, and environment problems
- `docs/deployment/render`
  Render-specific deployment notes
- `docs/deployment/hosting`
  Other hosting and deployment guides
- `docs/mobile`
  Flutter/mobile notes
- `docs/archive/reviews`
  Older review summaries and fix logs

For localhost web work, start with:

1. `README.md`
2. `QUICK_START_LOCAL.md`
3. `docs/setup/local/LOCAL_POSTGRESQL_SETUP.md`
4. `docs/setup/local/SETUP_VERIFICATION.md`

Current localhost email limitations are documented here:

- `docs/email/BREVO_EMAIL_DELIVERY_TROUBLESHOOTING.md`
- `docs/email/BREVO_API_SETUP.md`
